Instructions

Select the square which matches the RGB color stated in the game title. Here is what to look for:

The RGB color model adds different amounts (ranging from 0 - 255) of red, green and blue together to reproduce a broad array of colors.

  • The RGB primary colors are:
  • RGB(255, 0, 0): RED
  • RGB(0, 255, 0): GREEN
  • RGB(0, 0, 255): BLUE
  • The RGB secondary colors are:
  • RGB(255, 255, 0): YELLOW
  • RGB(0, 255, 255): CYAN
  • RGB(255, 0, 255): MAGENTA
